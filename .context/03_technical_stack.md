# Eldoraigne Tennis Club - Technical Stack

## Frontend & Framework
- **Language:** TypeScript (Strict Mode)
- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS (Brand Design tokens)
- **PWA:** Serwist (Offline caching & manifest)
- **Image Handling:** `browser-image-compression` (Client-side optimization)

## Backend & Database
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth (Google & Social Login)
- **Storage:** Google Drive API (For all club media and receipts)
- **Server Logic:** Next.js Server Actions

## Integrations & APIs
- **Database Backup/Mirror:** Google Sheets API
- **Document Management:** Google Drive API
- **Contact Management:** Google People API
- **Intelligence:** Google Gemini 1.5 Flash (via Vercel AI SDK)
- **Email:** Resend
- **WhatsApp:** 3rd Party Provider (QR Linkage)
- **Automation:** Vercel Cron Jobs (Scheduled tasks)

## Hosting & DevOps
- **Repository:** GitHub
- **Hosting:** Vercel (Serverless)
- **Environment:** Google Project IDX
